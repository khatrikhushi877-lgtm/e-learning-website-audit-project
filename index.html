Case Study: Coursera — Website performance audit, SEO analysis, UX review and digital marketing strategy.

Website Audit Core Web Vitals SEO Strategy Digital Marketing

Project Overview

This project analyzes Coursera as an e-learning platform and identifies areas where website speed, user experience, SEO and digital marketing can be improved. The audit was performed using Google PageSpeed Insights and supported with practical recommendations.

Website Selected

Coursera

https://www.coursera.org

Industry

Online Education / E-Learning

Main Focus

Performance, SEO, UX and Marketing

Problems Identified

  • Slow page loading on mobile devices.
  • High server response time.
  • Poor interaction responsiveness.
  • Heavy JavaScript affecting performance.
  • Homepage can be simplified for better user flow.

Recommended Improvements

  • Optimize Core Web Vitals.
  • Compress images and reduce unused scripts.
  • Improve caching and CDN delivery.
  • Strengthen internal linking and metadata.
  • Use clearer CTAs and cleaner homepage layout.

Digital Marketing Strategy

SEO

Keyword research, metadata, technical SEO and internal linking.

Content

Career blogs, course guides, learning tips and success stories.

Social Media

Instagram, LinkedIn, YouTube and student-focused content.

Paid Ads

Google Ads and Meta Ads targeting learners and professionals.

Conclusion

The audit shows that Coursera has a strong brand, professional design and good content structure. However, performance improvements such as better loading speed, lower server response time and optimized interaction responsiveness can improve user experience and search visibility.
